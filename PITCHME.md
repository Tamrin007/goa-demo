## goa とは

- マイクロサービス構築ためのフレームワーク
- API のデザインからモック・ドキュメントを自動生成
- API 開発者はビジネスロジックに集中できる
- クライアントサイド開発者は動くモックを素早く生成できる
- デザイン・ファースト
